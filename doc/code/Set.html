Class: Set

Set

new Set(l) → {Object}

A class to manipulate sets in Javascript. Note: Any function of this class, this constructor included, that takes a Set as parameter also accepts any object that can be turned into a set (e.g. Object, Array variables). For the sake of readability, this is not mentioned elsewhere.
Parameters:
Name Type Argument Description
l Set <optional>
 This optional parameter is an Array, a Set or an Object containing the elements to add to the newly created set.
